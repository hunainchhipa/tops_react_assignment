Q.3 What is JavaScript Output method?
Ans.
    JavaScript Output defines the ways to display the output of a given code. The output can be displayed by using four different ways which are listed below:

  1. innerHTML: The innerHTML property sets or returns the HTML content (inner HTML) of an element. It defines the HTML content.

    2. Syntax: document.getElementById("id").innerHTML;

  2. document.write(): The write() method writes directly to an open (HTML) document stream.

    3. Syntax: document.write()

  3. window.alert(): The alert() method is used when you want information to come through to the user.

    4. Syntax: window.alert()

  4. console.log(): The log() method writes a message to the console and it is used for debugging purposes.

    5. Syntax: console.log()

  5. window.prompt(): The prompt() method allows to take input from user.

    6. syntax: window.prompt()
